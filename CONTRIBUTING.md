# Contribution Guidelines

Thanks for helping improve this list. To keep it useful and trustworthy, please follow these rules.

## What belongs here

- Resources that genuinely help with decorating small, rented, or low-light homes.
- Free or mostly free tools, communities, guides, and shops. Paid resources only if they are clearly best in class and offer real free value.
- Resources that are active and maintained.

## What does not belong

- Affiliate or referral links.
- Self-promotion that adds no value to readers.
- Dead links, paywalled content with no free tier, or low-quality directories.

## How to contribute

1. Fork the repository and create a new branch.
2. Add your suggestion to the most fitting section, keeping the list alphabetical within reason.
3. Use this format, with a capitalised description that ends with a full stop:
   ```
   - [Name](https://example.com) - Short, specific description.
   ```
4. Check that the link works and the description is accurate and neutral.
5. Open a pull request with a short note on why the resource is useful.

## Quality bar

One excellent resource is worth more than ten average ones. If you are unsure whether something fits, open an issue to discuss it first.
